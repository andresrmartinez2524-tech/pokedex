# Pokedex Simple

Una Pokedex sencilla desarrollada con HTML, CSS y JavaScript que consume datos desde la API pública de PokéAPI.

El proyecto muestra varios Pokémon en tarjetas con su nombre, imagen, ID y tipo, y permite buscar Pokémon específicos mediante un input.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Fetch API
- PokéAPI

## Funcionalidades

- Mostrar múltiples Pokémon automáticamente
- Mostrar imagen del Pokémon
- Mostrar ID y tipos
- Buscador por nombre o número
- Manejo básico de errores

## API utilizada

https://pokeapi.co/

## Vista previa

La aplicación muestra tarjetas con:

- Nombre del Pokémon
- Imagen
- ID
- Tipo(s)

## Cómo usar el proyecto

1. Clonar el repositorio

git clone https://github.com/tu-usuario/pokedex.git

2. Abrir el archivo index.html en el navegador.

No requiere instalación de dependencias.

- Manejo de datos JSON
